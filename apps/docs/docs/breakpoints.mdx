# Breakpoints

## Example

```tsx
import { ChildPreview } from '@locospec/responsive-preview-react';

// Basic usage
<ChildPreview>
  <YourComponent />
</ChildPreview>

// With custom config
<ChildPreview
  config={{
    darkMode: true,
    showToolbar: true,
    showScale: true,
    showLabels: true
  }}
>
  <YourComponent />
</ChildPreview>
```
